Редакторы кода

  1. VS Code – мощный редактор (IDE)
  2. Sublime text 3 – удобный и быстрый редактор кода
  3. Atom – крутой редактор от GitHub
  4. Brackets – подходит для начинающих
  5. Notepad ++ – легендарный редактор
  6. WebStorm - довольно мощная, но платная IDE, имеющая пробный период в 30 дней.
    После окончания пробного периода редактор закрывается через каждые полчаса. Переустановите программу, чтобы сбросить пробные 30 дней.

    • AutoFileName и другие
